ubuntu时区设置

1. 查看可以设置的时区

ls -R --group-directories-first /usr/share/zoneinfo

中国就选择 Asia/Chongqing 既可以

2. 修改时区

timedatectl set-timezone Asia/Chongqing

ok date看一下时间正常了
